1. 如果冲高大跌发生后，第一段真实下跌后，中间走平，这个走平的样子非常重要，如果走平的时候，是倾斜向下的， 向上的绿线很少，这个让做空的人感觉很舒服。觉得它虽然下跌变慢，但是还在下跌。 那么它的后续下跌就不会很强，再次下跌发生后，会很短的跌两下，然后立刻反弹。 这时如果做空，应该立刻卖掉空仓。反手做多。或者寻找机会再次做空。因为这样的下跌表示，冲高大跌的走势 没有消耗掉。以后还会跌。只是暂时不跌了。

2. 如果冲高大跌发生后，中间走平的时候，反过来向上。虽然被拉回，但是还是动不动上冲。那么后面的大跌会很厉害。 这个走平会让做空的人很不舒服，觉得大盘随时要涨。

可持续性的真实上涨

B。 真实的上涨，上涨一段时间后，会走平很久。这时观察走平的样子，如果很平，或者似乎向下，那么会继续上涨。

-1. 陷阱大跌要么跌幅比较大，要么跌幅不大，但是在底部停留时间长，

0. 陷阱大跌之后的大涨，是否有走平非常重要。如果有上升一些，就有一段显著的走平，走平时间从5到20分钟不等， 那么上涨会很有力量。如果走平时间很短，或者走平次数不多，要警惕大盘很快就到达顶点。不过一般一个走平后， 至少会有有两个连续冲高。

1. 陷阱大跌的价值在于它后面出现的真实上涨，所以不要去占陷阱大跌的小便宜，真实的上涨前面都会示弱，要么刚刚开始涨就回调，然后 走平5分钟以上，要么稍微涨一些后，就走平10分钟以上。走平的形状显得要跌的样子，这样会让很多做多的人心存疑惑而退出。 也会让一些做空的人进来。如果没有走平，就连续大涨，如果超过两个台阶， 那么应该在第二个台阶退出，然后继续观察。可能是早熟的大涨。

2. 陷阱大跌发生的场景一定是前面小幅震荡，并没有大跌的样子，如果是从开盘一路大跌下来的，那么即使发生 类似的样子，那么大盘最后也会大跌的。这个可以参考下跌后反弹震荡走势 但是如果发生在下跌后半段，那么还是可能大幅反弹的

大涨时候误判陷阱大涨

2. 如果刚刚从陷阱大跌出来，第一个连续几分钟大涨。那么也不是

3. 涨幅必须很大。目前看10个点左右。

4. 如果是陷阱大涨，那么在高点不到20分钟就应该下跌，如果超过20分钟没有下跌，那么就不是。

陷阱大涨

1. 长时间震荡走平，第一分钟或者第二分钟上涨幅度就很大。那么这个就是陷阱大涨。这个一定要和真实上涨区别。

2. 和真实上涨的区别是，它前面一定有走平，然后第一分钟或者第二分钟涨幅很大。如果一直在持续上涨，突然幅度加大， 那么不是陷阱大涨，而是正常上涨。这种情况应该拿住多仓，大盘在加速上涨。但是也接近顶点。 虽然如此，也不能做空，因为不一定会跌。

前一天全幅度上涨

1. 前一天的涨幅远远超过普通的大涨。显得很强。

2. 第二天如果低开，那么会涨，如果高开，那么会在徘徊后，冲高。

3. 冲高后，会跌。可能走出幅度走势，或者冲高走平走势。

全力以赴低开走低反转走势

1. 前一天大跌或者连续几天大跌，今天低开走低
2. 低开走低后，要么幅度很大，接近最大可能低点，要么接近前一天跌幅
3. 寻找它见底的迹象，一旦发现，这个不会再回头，可以做多
4. 这个会开启连续大涨走势，未来一天或者几天可以逢低做多

1. 今天可能低开后，冲高，盘中到达高点后，回头，收盘到开盘点
2. 低开后，冲高大跌，然后震荡，一般收盘要么平盘，要么小幅上涨

连续大涨走势

连续大涨走势的操作：

a. 连续大涨走势很容易让人想做空，要坚决抵制这种想法
b. 连续三天大涨后，可能出现1%走势，或者高开走低再冲高走势
c. 某日显得很弱，比如低开后，小幅上涨，这时很容易让人觉得要做空 然而第二天就暴涨一天
d. 如果某天出现收盘远低于最高点，那么表示连续大涨走势结束了。
e. 如果某日低开，则坚决做多，前三天都会达到最大涨幅，收盘在最高点，要充分利用，
f. 连续大涨不一定天天涨，如果某天或者两天大盘收在接近高点，然而涨幅不大，那么第二天会高开暴涨，可以考虑事先买期权，

连续大涨走势的后续走势：

a. 连续大涨走势结束后，出现一天或者数日的盘整
b。再次开始上涨后，会连续上涨两天，这时如果第三天还涨，则会出现冲高走低走势

双底走势

1. 这个是指比较长时间的跌，包括冲高大跌，一般的低开后大跌。在长时间下跌的时候，大盘跌起来 似乎看不到尽头。反弹都不大。然后在低点某个点。出现比较大幅度的反弹。说明大盘到了底部。 这时经常会再次探底。可能和前面低点一样。也可能比前面低点要低几个点。这时不要贪婪。应该先买到再说。 要记住反弹要明显大于前面的反弹。而前面的反弹都很弱。

2. 双底不是总是出现。如果在底部反弹很急。出现两个台阶。每个台阶都涨幅比较大。那么下来的可能很大。 而且如果反弹前没有明显的加速动作。那么也会出现双底。而出现加速，就不会有双底了。

3. 如果不能确定大盘会双底，可以先做多call，等大盘反弹一定程度后，可以用期货做空对冲，可以在看到大盘 不会再下来的时候，平仓期货，或者一起平仓。

加速下跌见底

1. 这个长短时间下跌都有可能。

2. 先从高点下跌，下跌10，20几分钟后，反弹一些，然后突然一条红线， 幅度比较大，但是持续只有1分钟, 这个就是底部了。可以等2，3分钟，做多

3。只要不是下跌的第一阶段，那么一旦出现加速下跌后，就等2，3分钟就可以做多了

4. 下跌后，中间有一些走平，然后再次下跌。在某次走平时候，大盘加速下跌，这时在某个底部反弹，这时可以观察 它的底部的样子，如果缓慢反弹，那么就是真的底部，如果迅速反弹，但是很快又下来一下，虽然比前面的低点要高， 这个也可以看作是真实的大底。但是如果很快下来，又很快上去，那么可能未来还有更低的点。形成双底

A。迅速下跌后，在底部停留几分钟，缓慢反弹，后面加速反弹。
B。迅速下跌后，迅速反弹，反弹一定幅度后，在10分钟内，突然再次回头，但是最低点比上次高。
